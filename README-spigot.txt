[CENTER][SIZE=6][B][COLOR=#477a78]🧭 EnhancedCompass[/COLOR][/B][/SIZE][/CENTER]

[CENTER][I]A powerful and intuitive compass plugin for Minecraft that transforms your navigation experience.[/I][/CENTER]

[SIZE=5][B]✨ Features[/B][/SIZE]

[LIST]
[*]🎯 [B]Point compass to players[/B] - Navigate to any online player instantly
[*]📍 [B]Save custom locations[/B] - Create personal and global waypoints
[*]🔢 [B]Navigate by coordinates[/B] - Jump to any XYZ position
[*]👥 [B]Group-based location limits[/B] - Configurable limits per permission group
[*]🌐 [B]Multi-language support[/B] - Localized messages for your community
[*]⚡ [B]High performance[/B] - Lightweight and optimized for busy servers
[/LIST]

[SIZE=5][B]🚀 Quick Start[/B][/SIZE]

[B]Installation:[/B]
[LIST=1]
[*]Download the latest release
[*]Drop the JAR file into your [ICODE]plugins/[/ICODE] directory
[*]Restart your server
[*]Configure groups and permissions (optional)
[/LIST]

[B]Basic Commands:[/B]

[LIST]
[*][B]Point to player Steve[/B] - /compass Steve           
[*][B]Point to saved location "home"[/B] - /compass home           
[*][B]Point to coordinates[/B] - /compass 100 64 200     
[*][B]Save current location[/B] - /compass set home       
[*][B]Reset to world spawn[/B] - /compass reset          
[/LIST]
